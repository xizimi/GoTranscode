# GoTranscode - 分布式视频转码系统

基于 Go 语言开发的高性能分布式视频转码系统，支持多节点协同工作、任务队列管理、实时监控和自适应码率转码。

## 功能特性

- **分布式架构**：支持多节点水平扩展，通过 RabbitMQ 实现任务分发
- **智能转码**：基于 CQP 探针的视频复杂度分析，动态调整 CRF 参数
- **多档位输出**：支持 360p/480p/720p/1080p 多分辨率转码
- **HLS 流媒体**：自动生成 Master Playlist，支持自适应码率切换
- **优先级队列**：VIP 任务优先处理，支持任务优先级调度
- **失败重试**：死信队列机制，支持最多 3 次自动重试
- **实时监控**：Prometheus 指标采集，支持 CPU/内存/任务统计监控
- **高可用性**：节点心跳机制，Redis 状态持久化，优雅关闭
- **任务管理**：完整的任务生命周期管理（pending/running/completed/failed）
